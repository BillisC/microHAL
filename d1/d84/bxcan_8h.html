void bxcan_configure_bitrate ( const struct bxCANBitrateConfig config)

Configures the bxCAN bitrate according to config.

The configuration options for the baudrate is located in the bx for CAN are specified in the bxCANBitrateConfig struct. Make sure to not exceed the bit limit of the values and calculate the expected outcome beforehand.
